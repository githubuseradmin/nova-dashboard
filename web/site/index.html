Full-stack starter

A clean, modern full-stack starter.

A landing page, session-based auth, a user dashboard and an admin panel — built on Go, Svelte and PostgreSQL, with a bespoke UI and no framework bloat.

Open the dashboard → View source
  • Go · net/http · chi
  • PostgreSQL · pgx
  • Svelte 5 · TypeScript · Vite
  • argon2id · sessions · CSRF
  • Nginx · Certbot

Native backend

Standard-library net/http with the chi router — no fasthttp, no heavy framework. Just explicit, idiomatic Go.

Explicit data layer

pgx with hand-written SQL and migrations. No ORM magic — you always know the query that runs.

Tiny frontend

Svelte 5 compiles to a ~17 kB gzipped bundle. A bespoke design system in plain CSS with light/dark — no Tailwind, no UI kit.

Secure by default

argon2id password hashing, opaque server-side sessions in HttpOnly cookies, CSRF protection and role-based access.
